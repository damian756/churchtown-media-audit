# Churchtown Media

**High-performance web design and technical SEO agency based in Southport, Merseyside.**

Live site: [churchtownmedia.co.uk](https://www.churchtownmedia.co.uk)

---

## What This Is

The main Churchtown Media agency site. Built to demonstrate the same stack and performance standards used for all client and network projects. Scores 95+ on PageSpeed Insights. Zero unnecessary dependencies.

## Stack

- **Framework**: Next.js (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **Deployment**: Vercel
- **Analytics**: Plausible (cookieless, GDPR-compliant)

## Features

- Static generation throughout — no client-side data fetching where unnecessary
- Full Schema.org JSON-LD (Organization, Person, LocalBusiness, Article)
- Blog with structured data and E-E-A-T signals
- Core Web Vitals optimised — LCP, CLS, INP all green
- Canonical URLs, OpenGraph, Twitter cards on every page
